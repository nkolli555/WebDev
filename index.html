We are going to learn below 3 topics in our Web Development course :

  1. HTML -> Hyper Text Markup Language is used to define 'what' content to be displayed inside web pages like text, images, etc.,

  2. CSS -> Cascading Style Sheet is used to 'how' content to be displayed inside web pages like color, size, placement, etc.,

  3. JS -> Java Script is used to make web pages dynamic like submit button
